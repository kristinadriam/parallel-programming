# ДЗ: Клиентская нода подбора паролей

Консольное приложение на C++: подключается по TCP к управляющему серверу, получает диапазон кандидатов и целевой хэш, параллельно перебирает ключи и отправляет результат.

## Сборка и запуск

```bash
make
./app <host> <port> <путь_к_файлу_hashrate>
```

В файле hashrate — одно целое число: оценка числа вычислений `sha1(md5(key))` в секунду (для сервера).

## Проверка

`test_server.py` — простой сервер на Python с той же схемой хэша, что и клиент:

```bash
python3 test_server.py 12345
./app 127.0.0.1 12345 hashrate.txt
```
