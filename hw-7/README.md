# ДЗ-7: суммирование матрицы по строкам и столбцам

## что делаем

есть матрица 8000×8000. суммируем все ее элементы двумя способами:

- **по строкам** — внешний цикл по строкам, внутренний по столбцам.
- **по столбцам** — внешний цикл по столбцам, внутренний по строкам.

измеряем время и аппаратные счетчики через `mperf-stat`.

## почему столбцы медленнее

все дело в кэше. матрица хранится построчно — элементы одной строки лежат в памяти рядом. когда процессор загружает один элемент, он затаскивает в кэш-линию (64 байта) сразу 16 соседних `int`-ов.

**по строкам** — идем подряд по памяти: загрузили кэш-линию, 16 элементов из нее прочитали бесплатно. промахов мало (~8.5 млн).

**по столбцам** — на каждом шаге прыгаем в другую строку, которая лежит в совершенно другом месте памяти. загруженная кэш-линия бесполезна — следующий элемент из нее нам не нужен. промахов в 9 раз больше (~80 млн), время в 25 раз больше (145 мс против 5.8 мс).

## что за оптимизация

**для столбцов:** вместо того чтобы прыгать по строкам, обходим матрицу по строкам (как в sumRows), но накапливаем суммы каждого столбца в отдельный массив `colSums[j]`. потом складываем этот массив. доступ к памяти стал последовательным — промахи упали с 80 млн до 6.4 млн, время — со 145 мс до 8 мс.

```cpp
int64 sumColsOptimized(const Matrix& m) {
    int n = m.size();
    std::vector<int64> colSums(n, 0);
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < N; ++j) {
            colSums[j] += m[i][j];
        }
    }
    int64 total = 0;
    for (int j = 0; j < n; ++j) {
        total += colSums[j];
    }
    return total;
}
```

**для строк:** развертка цикла с 4 аккумуляторами, чтобы процессор мог выполнять сложения параллельно. эффект небольшой (5.79 → 5.64 мс), потому что построчный обход и так быстрый.

## вывод

порядок обхода данных критически влияет на производительность из-за кэша. суммирование по столбцам в 25 раз медленнее, чем по строкам, хотя работы одинаково. если изменить порядок доступа к памяти на последовательный, разница почти исчезает.
